# Rahul Sharma - Personal Portfolio

## Overview
Personal portfolio website showcasing my journey as an Aspiring DevOps Engineer, built with pure HTML, CSS, and JavaScript.

## Features
- Responsive design
- Single-page application
- Sections:
  - About
  - Experience
  - Projects
  - Contact

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Setup and Installation
1. Clone the repository
```bash
git clone https://github.com/Rahul0972/portfolio.git
```
2. Open `index.html` in your browser

## Deployment
This portfolio is deployed using GitHub Pages. You can access it at: https://rahul0972.github.io/portfolio/
